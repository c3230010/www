Box Design Practice

Box1

This is the first box. It has a light blue background. rounded corners, and a solid border. This box includes an image below to
enhance the visual appeal of the contest. The image should be relevant to the contest described and help illustrate the point.

Box2

This is the second box. It has a light orange background, slightly rounded corners, a solid border, and a shadow. includind an
image in this box break up the text and provides a visual example of the contest. The use of CSS box shadow enhances the
3D effect of this box.

Box3

This is the third box. It has a light purple background. no rounded corners, and a dashed border. This box also includes an image,
which helps to demonstrate how different styles and borders can be applied to create distinct visual sections within a webpage.
